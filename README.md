# CustomTkinter Password Generator 🔐

A modern desktop application built with Python and CustomTkinter that generates secure passwords based on user preferences.

## ✨ Features
*   **Modern UI:** Built using CustomTkinter with a sleek dark/green hacker theme.
*   **Custom Length:** Adjust password length from 8 to 24 characters using a slider.
*   **Flexible Parameters:** Include/exclude digits, letters, and special characters.
*   **Clipboard Integration:** Copy your generated password with a single click (includes visual success feedback without UI freezing).

## 🛠️ Requirements & Installation

1. Install Python 3.x
2. Install CustomTkinter library:
   ```bash
   pip install customtkinter
   ```
3. Run the application
